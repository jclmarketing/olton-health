# Olton Health & Performance — Modern Site

Modernised, SEO-optimised replacement for [oltonhealth.com](https://www.oltonhealth.com). Static site with 51 pages.

## Structure

- **6 service pages** — osteopathy, physiotherapy, sports massage, medical acupuncture, Pilates, strength & conditioning
- **27 local area pages** — Olton, Solihull, Shirley, Knowle, Dorridge, Acocks Green, Hall Green, Yardley, Sheldon and more
- **6 condition pages** — back pain, neck pain, sciatica, sports injuries, knee pain, shoulder pain
- **Supporting** — home, about, team, contact, book, testimonials, FAQs, privacy, 404
- **SEO** — sitemap.xml, robots.txt, MedicalBusiness/LocalBusiness JSON-LD, FAQ schema, MedicalProcedure schema

## Build

```
node build.js
```

Pages are generated from `data.js` + `templates.js` into static HTML.

## Stack

Static HTML/CSS/JS — no framework. Plus Jakarta Sans + Fraunces. Custom CSS, no Tailwind/Bootstrap.

## Brand

- Navy `#1A2A6E`
- Crimson `#8B1F2A`
- Sky `#B5DCEA`
- Logo and clinic photography preserved from existing site.
